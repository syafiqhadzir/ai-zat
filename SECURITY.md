# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** report it via public GitHub issues.

Instead, please email **info@ikatan.my** with the details. We will review the issue and address it promptly.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |
